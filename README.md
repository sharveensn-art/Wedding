# Wedding Decoration Preview

An interactive 3D decoration preview for a Hindu temple wedding ceremony.

## Files

| File | Description |
|---|---|
| `temple_3d.html` | Interactive 3D temple scene — orbit, pan, zoom, multiple camera views |
| `decoration_plan.html` | Decoration mood board & written plan |

## How to View

Open either file in **Chrome or Edge**. Requires an internet connection (loads Three.js from CDN).

### Controls
- **Left drag** — Orbit around the scene
- **Right drag** — Pan
- **Scroll** — Zoom in / out
- **View buttons** — Jump to Overview · Entrance · Aisle · Mandap · Top Down

## Color Theme
Serene & minimalistic — deep red roses, white jasmine, brass/gold, warm ivory.
Designed to complement the bride's deep crimson saree.
